# /start — Start both servers

Start the Spring Boot backend and React frontend dev servers for this project.

1. In one terminal, run `mvn spring-boot:run` from the project root to start the backend on http://localhost:8080
2. In another terminal, run `npm run dev` from the `frontend/` directory to start the frontend on http://localhost:5173

Confirm both servers are running and report their URLs. If either fails, show the error output and suggest a fix.
